C# Introduction

  • C# is a modern, object-oriented programming language developed by Microsoft as part of the .NET framework.
  • C# is widely used for building desktop applications, web applications, and game development (especially with Unity).
  • It is known for its simplicity, scalability, and performance.

Why Learn C#?

  • Versatility: C# can be used for a variety of applications, from web to game development.
  • Rich Libraries: The .NET Framework provides a rich set of libraries for building software solutions.
  • Strong Object-Oriented Features: C# is a fully object-oriented language, making it easy to structure large-scale applications.
  • Community and Support: Being backed by Microsoft, C# has a large developer community and comprehensive documentation.

C# Data Types

C# has a rich set of data types to support various operations. These data types can be classified into value types and reference types.


Value Types:

  • int: Used to store integer values.
  • float: Used to store floating-point numbers.
  • double: Used to store double-precision floating-point numbers.
  • char: Used to store a single character.
  • bool: Used to store Boolean values (true or false).

Reference Types:

  • string: Used to store sequences of characters.
  • array: Used to store a collection of elements of the same type.
  • class: Used to define objects.

Example:


                    int age = 30;
                    float height = 5.9f;
                    bool isAdult = true;
                    string name = "John";

C# Operators

C# includes a wide range of operators, including arithmetic, relational, logical, and bitwise operators.


Arithmetic Operators:

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division
  • %: Modulus (remainder)

Relational Operators:

  • ==: Equal to
  • !=: Not equal to
  • >: Greater than
  • <: Less than

Example:


                    int a = 10, b = 20;
                    if(a < b) {
                        Console.WriteLine("a is less than b");
                    }

C# Booleans

The bool type in C# can only hold two values: true or false.


Boolean Expressions:

  • Boolean values are typically used in control flow statements such as if, while, and for loops.
  • Boolean expressions can be combined with logical operators like && (AND), || (OR), and ! (NOT).

Example:


                    bool isRaining = false;
                    if(isRaining) {
                        Console.WriteLine("Take an umbrella!");
                    } else {
                        Console.WriteLine("Enjoy the sunshine!");
                    }

C# If ... Else

The if statement is used to execute a block of code based on a condition. The else statement is used as a fallback if the condition is false.


Syntax:


                    if(condition) {
                        // Code to execute if the condition is true
                    } else {
                        // Code to execute if the condition is false
                    }

Example:


                    int age = 18;
                    if(age >= 18) {
                        Console.WriteLine("You are an adult.");
                    } else {
                        Console.WriteLine("You are a minor.");
                    }

C# While Loop

The while loop repeatedly executes a block of code as long as the specified condition evaluates to true.


Syntax:


                    while(condition) {
                        // Code to execute
                    }

Example:


                    int i = 0;
                    while(i < 5) {
                        Console.WriteLine(i);
                        i++;
                    }

C# For Loop

The for loop allows you to execute a block of code a specified number of times.


Syntax:


                    for(initialization; condition; increment/decrement) {
                        continue;  // Skips when i is 5
                        // Code to execute
                    }

Example:


                    for(int i = 0; i < 5; i++) {
                        Console.WriteLine(i);
                    }

C# Break and Continue

The break statement is used to exit a loop prematurely, and the continue statement is used to skip the current iteration and proceed to the next one.


Example with break:


                for(int i = 0; i < 10; i++) {
                    if(i == 5) {
                        break;  // Exits the loop when i is 5
                    }
                    Console.WriteLine(i);
                }

Example with continue:


                for(int i = 0; i < 10; i++) {
                    if(i == 5) {
                        

                        }
                        Console.WriteLine(i);
                    }

C# Lists

In C#, a list is a generic collection of objects that can be resized dynamically.


List Definition:


                  using System;
                  using System.Collections.Generic;
                   List fruits = new List();
                  fruits.Add("Apple");
                  fruits.Add("Banana");
                  fruits.Add("Cherry");
                  Console.WriteLine(fruits[0]);  // Output: Apple

Common Methods:

  • Add(): Adds an element to the list.
  • Remove(): Removes the first occurrence of a specific object from the list.
  • Count: Gets the number of elements in the list.
  • Contains(): Checks if an element exists in the list.

C# Strings

Strings in C# are immutable sequences of characters. Once a string is created, it cannot be changed.


String Declaration:


                  string greeting = "Hello, World!";
                  Console.WriteLine(greeting);  // Output: Hello, World!

Common String Methods:

  • Length: Gets the length of the string.
  • Substring(): Extracts a substring from a string.
  • ToUpper(): Converts the string to uppercase.
  • ToLower(): Converts the string to lowercase.
  • Contains(): Checks if a substring is contained within the string.

Example:


                  string name = "John";
                  string upperName = name.ToUpper();  // JOHN
                  string lowerName = name.ToLower();  // john

C# User Input

In C#, user input is commonly handled using the Console.ReadLine() method, which reads a line of text from the console.


Example - Getting User Input:


                  Console.WriteLine("Enter your name:");
                  string name = Console.ReadLine();
                  Console.WriteLine("Hello, " + name + "!");

Converting Input to Specific Types:

  • int.Parse(): Converts the input string to an integer.
  • double.Parse(): Converts the input string to a double.
  • Convert.ToInt32(): Converts the input string to an integer (handles nulls more safely).

Example - Converting Input:


                  Console.WriteLine("Enter your age:");
                  int age = int.Parse(Console.ReadLine());
                  Console.WriteLine("Your age is: " + age);

C# Functions

In C#, a function is a block of code that performs a specific task. Functions (or methods) are defined inside classes and can take parameters and return values.


Function Definition:


                  using System;
                   public class Program
                  {
                      public static int AddNumbers(int a, int b)
                      {
                          return a + b;
                      }
                       public static void Main(string[] args)
                      {
                          int result = AddNumbers(5, 10);
                          Console.WriteLine("Sum: " + result);  // Output: Sum: 15
                      }
                  }

Function Types:

  • void: A function that does not return any value.
  • Return Type: A function that returns a value (e.g., int, string).

Example - Void Function:


                  public static void GreetUser(string name)
                  {
                      Console.WriteLine("Hello, " + name + "!");
                  }

C# Dictionaries

A Dictionary in C# is a collection of key-value pairs, where each key is unique. It provides fast lookups based on the key.


Dictionary Declaration:


                  using System;
                  using System.Collections.Generic;
                   Dictionary ageDictionary = new Dictionary();
                  ageDictionary.Add("John", 30);
                  ageDictionary.Add("Alice", 25);
                  ageDictionary.Add("Bob", 35);
                   Console.WriteLine(ageDictionary["John"]);  // Output: 30

Common Dictionary Methods:

  • Add(): Adds a key-value pair to the dictionary.
  • ContainsKey(): Checks if a key exists in the dictionary.
  • Remove(): Removes the key-value pair by key.
  • Keys: Gets a collection of keys in the dictionary.
  • Values: Gets a collection of values in the dictionary.

C# File I/O

C# provides several classes in the System.IO namespace to work with files, including reading, writing, and manipulating files.


Reading from a File:


                  using System;
                  using System.IO;
                   string filePath = @"C:\path\to\file.txt";
                  string fileContent = File.ReadAllText(filePath);
                  Console.WriteLine(fileContent);

Writing to a File:


                  string content = "Hello, World!";
                  File.WriteAllText(@"C:\path\to\file.txt", content);

Other File I/O Methods:

  • File.Exists(): Checks if the file exists.
  • File.AppendAllText(): Appends text to the end of a file.
  • StreamReader: Provides a way to read characters from a byte stream.
  • StreamWriter: Provides a way to write characters to a byte stream.

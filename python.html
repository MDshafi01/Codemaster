Python Introduction

  • Python is a high-level, interpreted programming language developed by Guido van Rossum and released in 1991.
  • Python is known for its simplicity and readability, making it an ideal choice for beginners and experts alike.
  • Python is used in a variety of domains, including web development, data science, artificial intelligence, scientific computing, and more.

Why Learn Python?

  • Ease of Learning: Python's syntax is simple and easy to understand.
  • Extensive Libraries: Python has a wide range of libraries and frameworks for various domains, including web development (Django, Flask), data science (Pandas, NumPy), and more.
  • Versatility: Python can be used for web development, machine learning, automation, and more.
  • Strong Community: Python has an active community of developers, which makes it easier to get help and support.

Data Types in Python

Python supports several data types, including:


Basic Data Types:

  • int: Integer type, used to represent whole numbers.
  • float: Floating point type, used to represent decimal numbers.
  • str: String type, used to represent text.
  • bool: Boolean type, used to represent True or False values.
  • list: A collection of ordered, mutable elements.
  • tuple: An immutable collection of ordered elements.
  • dict: Dictionary type, used to store key-value pairs.

Example:


                    age = 25
                    height = 5.9
                    name = "John"
                    is_adult = True
                    fruits = ["apple", "banana", "cherry"]

Python Operators

Python supports various types of operators:


Types of Operators:

  • Arithmetic Operators: +, -, *, /, %, //, **
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: and, or, not
  • Assignment Operators: =, +=, -=, *=, /=
  • Membership Operators: in, not in

Booleans in Python

In Python, the boolean type represents the two truth values: True and False.

Example:


                    is_even = True
                    is_odd = False

                    if is_even:
                        print("The number is even.")
                    else:
                        print("The number is odd.")

Python If ... Else

In Python, you can use if, elif, and else to make decisions in your programs.


Syntax:


                    if condition:
                        # Code to execute if the condition is True
                    elif another_condition:
                        # Code to execute if the second condition is True
                    else:
                        # Code to execute if no condition is True

Example:


                    age = 18
                    if age >= 18:
                        print("You are an adult.")
                    else:
                        print("You are a minor.")

Python While Loop

While loops in Python are used to execute a block of code repeatedly as long as a given condition is True.


Syntax:


                    while condition:
                        # Code to execute

Example:


                    count = 0
                    while count < 5:
                        print("Count:", count)
                        count += 1

Python Break and Continue

Python provides the break and continue statements to control the flow of loops.


break Statement:

The break statement is used to exit the current loop prematurely.


continue Statement:

The continue statement is used to skip the rest of the code inside a loop for the current iteration and move to the next iteration.


Example:


                    for i in range(1, 6):
                        if i == 3:
                            continue  # Skip the rest of the loop when i is 3
                        print(i)
                    # Output: 1, 2, 4, 5

Python Lists

A list in Python is a mutable, ordered collection of elements. You can store multiple items in a single variable using lists.


Creating a List:


                    fruits = ["apple", "banana", "cherry"]
                    numbers = [1, 2, 3, 4, 5]

List Operations:

  • append(x): Adds an item to the end of the list.
  • remove(x): Removes the first occurrence of the item from the list.
  • len(): Returns the number of items in the list.
  • sort(): Sorts the list in ascending order.

Example:


                    fruits = ["apple", "banana", "cherry"]
                    fruits.append("orange")  # Adds "orange" to the list
                    print(fruits)
                    # Output: ['apple', 'banana', 'cherry', 'orange']

Python Strings

In Python, strings are sequences of characters. Strings are immutable, meaning you cannot modify a string after it's created.


Creating a String:


                    greeting = "Hello, World!"
                    name = 'John'

String Operations:

  • len(): Returns the length of the string.
  • upper(): Converts all characters to uppercase.
  • lower(): Converts all characters to lowercase.
  • strip(): Removes whitespace from both ends of the string.

Example:


                    greeting = "  Hello, World!  "
                    print(greeting.strip())  # Output: 'Hello, World!'

Python User Input

Python provides the input() function to allow the user to enter data during the execution of the program.


Syntax:


                    user_input = input("Enter your name: ")
                    print("Hello, " + user_input)

Example:


                    name = input("Enter your name: ")
                    age = input("Enter your age: ")
                    print(f"Hello {name}, you are {age} years old.")

Python Functions

Functions in Python are defined using the def keyword. Functions allow you to group a block of code together and reuse it.


Syntax:


                    def function_name():
                        # Code to be executed

Example:


                    def greet(name):
                        print(f"Hello, {name}!")
                    
                    greet("Alice")  # Output: Hello, Alice!

Python Dictionaries

A dictionary in Python is an unordered collection of key-value pairs.


Creating a Dictionary:


                    person = {"name": "John", "age": 30, "city": "New York"}

Dictionary Operations:

  • keys(): Returns a view object containing the keys of the dictionary.
  • values(): Returns a view object containing the values of the dictionary.
  • get(): Returns the value associated with the given key.

Example:


                    person = {"name": "John", "age": 30, "city": "New York"}
                    print(person["name"])  # Output: John

Python File I/O

Python provides functions to work with files. You can read from and write to files using built-in functions like open(), read(), and write().


Opening a File:


                    file = open("example.txt", "r")  # Open file in read mode

Reading from a File:


                    content = file.read()  # Reads the entire content of the file
                    print(content)

Writing to a File:


                    file = open("example.txt", "w")  # Open file in write mode
                    file.write("Hello, World!")  # Writes to the file
                    file.close()

Python Classes and Objects

Python is an object-oriented programming language. You can define classes to create objects, which are instances of those classes.


Creating a Class:


                    class Person:
                        def __init__(self, name, age):
                            self.name = name
                            self.age = age
                        
                        def greet(self):
                            print(f"Hello, my name is {self.name} and I am {self.age} years old.")
                    
                    person1 = Person("Alice", 25)
                    person1.greet()  # Output: Hello, my name is Alice and I am 25 years old.

Java Introduction


Java is a high-level, object-oriented programming language that was designed to have as few implementation dependencies as possible. It was developed by Sun Microsystems in 1995 and is now owned by Oracle.


Why Learn Java?

  • Platform Independence: Java programs can run on any device that has the Java Virtual Machine (JVM) installed, making it platform-independent.
  • Object-Oriented: Java is fully object-oriented, meaning it uses classes and objects to organize the code and make it reusable.
  • Wide Usage: Java is widely used in enterprise applications, Android development, and web development.
  • Large Community & Libraries: Java has a large developer community and a vast ecosystem of libraries and frameworks.

Java Syntax


The syntax of Java is similar to C and C++, making it easier for those familiar with these languages to learn Java. Here's the basic structure of a Java program:


Basic Structure:


                public class Main {
                    public static void main(String[] args) {
                        int a = 5, b = 10;
                        int sum = a + b;
                        System.out.println("Sum: " + sum);
                    }
                }

Explanation:

  • public class Main: This defines the class named `Main`. In Java, every program must have a class.
  • public static void main(String[] args): This is the entry point of any Java program. The execution of the program starts here.
  • System.out.println: This is used to print output to the console in Java.

Data Types in Java


Java provides several data types, including primitive types and reference types.


Primitive Data Types:

  • int: Used to store integers (whole numbers).
  • float: Used to store decimal numbers (single-precision).
  • double: Used to store larger decimal numbers (double-precision).
  • char: Used to store a single character.
  • boolean: Used to store true or false values.

Example:


                int age = 30;
                float temperature = 36.6f;
                double price = 99.99;
                char grade = 'A';
                boolean isJavaFun = true;

Booleans in Java


In Java, the boolean data type is used to represent logical values. It can only hold one of two possible values: true or false. Booleans are commonly used in conditional statements like if, while, and for loops to control the flow of the program based on certain conditions.


Boolean Values:

  • true: Represents a true condition or statement.
  • false: Represents a false condition or statement.

Example of Boolean in Java:


                boolean isAdult = true;
                boolean hasLicense = false;
        
                if (isAdult && hasLicense) {
                    System.out.println("You can drive.");
                } else {
                    System.out.println("You cannot drive.");
                }
                // Output: "You cannot drive."

Boolean Expression Example:


In the example above, the condition isAdult && hasLicense is evaluated as a boolean expression. The operator && (AND) returns true only if both conditions are true. In this case, hasLicense is false, so the overall expression evaluates to false.


Java Operators


Java supports various types of operators:


Types of Operators:

  • Arithmetic Operators: +, -, *, /, %
    • Used to perform arithmetic operations.
  • Relational Operators: ==, !=, >, <, >=, <=
    • Used to compare two values.
  • Logical Operators: &&, ||, !
    • Used to combine multiple conditions.
  • Assignment Operators: =, +=, -=, *=, /=
    • Used to assign values to variables.

Java If ... Else


Similar to C, Java uses if, else if, and else to make decisions based on conditions.


Syntax:


                if (condition) {
                    // block of code to be executed if condition is true
                } else if (condition2) {
                    // block of code to be executed if condition2 is true
                } else {
                    // block of code to be executed if none of the conditions are true
                }

Example:


                int time = 20;
                if (time < 18) {
                    System.out.println("Good day.");
                } else {
                    System.out.println("Good evening.");
                }
                // Outputs "Good evening."

Java Switch


The switch statement is used to select one of many code blocks to be executed.


Syntax:


                switch (expression) {
                    case value1:
                        // block of code
                        break;
                    case value2:
                        // block of code
                        break;
                    default:
                        // block of code
                }

Example:


                int day = 2;
                switch (day) {
                    case 1:
                        System.out.println("Sunday");
                        break;
                    case 2:
                        System.out.println("Monday");
                        break;
                    case 3:
                        System.out.println("Tuesday");
                        break;
                    default:
                        System.out.println("Invalid day");
                }
                // Outputs "Monday"

Java While Loop


The `while` loop in Java is used to execute a block of code as long as the specified condition is true.


Syntax:


                while (condition) {
                    // block of code to be executed
                }

Example:


                int i = 0;
                while (i < 5) {
                    System.out.println(i);
                    i++;
                }
                // Outputs: 0, 1, 2, 3, 4

Java For Loop


The `for` loop is used to execute a block of code a specific number of times. It's ideal when you know in advance how many times you want to execute a statement or a block of statements.


Syntax:


                for (initialization; condition; increment/decrement) {
                    // block of code to be executed
                }

Example:


                for (int i = 0; i < 5; i++) {
                    System.out.println(i);
                }
                // Outputs: 0, 1, 2, 3, 4

Java Break and Continue


The `break` and `continue` statements are used to control the flow of loops in Java. The `break` statement is used to exit the loop entirely, while the `continue` statement is used to skip the current iteration and proceed with the next iteration.


Syntax:


                break;   // exits the loop
                continue; // skips the current iteration

Example:


                for (int i = 0; i < 5; i++) {
                    if (i == 2) {
                        continue; // Skip when i == 2
                    }
                    System.out.println(i);
                }
                // Outputs: 0, 1, 3, 4

Java Arrays


Arrays in Java are used to store multiple values in a single variable, instead of declaring individual variables for each value.


Syntax:


                dataType[] arrayName = new dataType[size];

Example:


                int[] numbers = {1, 2, 3, 4, 5};
                System.out.println(numbers[0]); // Outputs: 1

Java Strings


In Java, a string is an object that represents a sequence of characters. The `String` class is used to work with strings in Java.


Syntax:


                String str = "Hello, World!";

Example:


                String message = "Java Programming";
                System.out.println(message.length()); // Outputs: 17
                System.out.println(message.toUpperCase()); // Outputs: "JAVA PROGRAMMING"

Java User Input


In Java, user input can be taken using the `Scanner` class from the `java.util` package. This allows you to get input from the user during the execution of the program.


Syntax:


                Scanner scanner = new Scanner(System.in);
                String input = scanner.nextLine(); // For string input
                int number = scanner.nextInt();    // For integer input

Example:


                Scanner scanner = new Scanner(System.in);
                System.out.print("Enter your name: ");
                String name = scanner.nextLine();
                System.out.println("Hello, " + name);

Java Memory Address


In Java, direct memory access is not allowed. Java provides automatic memory management through garbage collection, meaning that the programmer does not have to manually allocate or free memory as in languages like C or C++.


Although Java does not use explicit memory addresses, the Java Virtual Machine (JVM) internally handles memory allocation and deallocation. This means you don't need to worry about memory management, which helps prevent memory-related issues like memory leaks.


Java Pointers


Unlike languages such as C and C++, Java does not support pointers directly. In Java, we use references to objects instead of pointers, which makes memory management easier and safer.


Although references behave similarly to pointers (they point to objects in memory), Java's reference system ensures that memory is automatically managed by the JVM. This reduces common issues with pointers, such as pointer arithmetic errors and null pointer dereferencing.


Example:


                class Person {
                    String name;
                    int age;
                }
        
                Person person1 = new Person();
                person1.name = "John";
                person1.age = 30;
        
                Person person2 = person1; // person2 references the same object as person1
                System.out.println(person2.name); // Outputs: John

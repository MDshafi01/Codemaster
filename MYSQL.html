MySQL Introduction

  • MySQL is an open-source relational database management system (RDBMS) based on Structured Query Language (SQL).
  • It is used for storing and managing data in a tabular format, where data is organized into rows and columns.
  • MySQL is widely used in web applications, especially with PHP, for building dynamic websites and applications.

Why Learn MySQL?

  • Speed: MySQL is known for its fast performance when querying large datasets.
  • Reliability: It is reliable, with ACID-compliant features to handle complex database operations.
  • Scalability: MySQL can scale to support massive databases and handle high-traffic applications.
  • Community Support: MySQL has a large and active community offering help, tutorials, and resources.

MySQL Data Types

MySQL supports various data types for storing different kinds of data in a database. These can be classified into numeric, date/time, and string types.


Numeric Types:

  • INT: Used to store integer values.
  • DECIMAL: Stores exact numeric values with fixed decimal points.
  • FLOAT: Stores floating-point numbers.
  • DOUBLE: Stores double-precision floating-point numbers.

Date and Time Types:

  • DATE: Stores a date value in the format YYYY-MM-DD.
  • DATETIME: Stores both date and time values.
  • TIME: Stores time values in the format HH:MM:SS.
  • YEAR: Stores a 4-digit year.

String Types:

  • VARCHAR: Stores variable-length strings.
  • CHAR: Stores fixed-length strings.
  • TEXT: Stores long text strings, up to 65,535 characters.
  • BLOB: Used to store binary large objects.

Example:


                    CREATE TABLE users (
                        id INT AUTO_INCREMENT PRIMARY KEY,
                        name VARCHAR(100),
                        email VARCHAR(100),
                        age INT
                    );

MySQL Operators

Operators in MySQL are used to perform operations on data. These include arithmetic, comparison, and logical operators.


Arithmetic Operators:

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division
  • %: Modulus

Comparison Operators:

  • ==: Equal to
  • !=: Not equal to
  • >: Greater than
  • <: Less than
  • >=: Greater than or equal to
  • <=: Less than or equal to

Logical Operators:

  • AND: Logical AND
  • OR: Logical OR
  • NOT: Logical NOT

Example:


                    SELECT * FROM users WHERE age > 20 AND email LIKE '%@gmail.com';

MySQL Constraints

Constraints in MySQL are used to limit the type of data that can be inserted into a table. Some common constraints are:


  • PRIMARY KEY: Uniquely identifies each record in a table.
  • FOREIGN KEY: Ensures referential integrity between two tables.
  • UNIQUE: Ensures that all values in a column are different.
  • NOT NULL: Ensures that a column cannot have NULL values.
  • DEFAULT: Sets a default value for a column if no value is provided.

Example:


                    CREATE TABLE users (
                        id INT AUTO_INCREMENT PRIMARY KEY,
                        name VARCHAR(100) NOT NULL,
                        email VARCHAR(100) UNIQUE,
                        age INT DEFAULT 18
                    );

MySQL SELECT

The SELECT statement is used to retrieve data from a database. It is one of the most common SQL operations.


Syntax:


                    SELECT column1, column2 FROM table_name WHERE condition;

Example:


                    SELECT * FROM users;

MySQL INSERT

The INSERT INTO statement is used to insert new records into a table.


Syntax:


                    INSERT INTO table_name (column1, column2)
                    VALUES (value1, value2);

Example:


                    INSERT INTO users (name, email, age)
                    VALUES ('John Doe', 'john@example.com', 30);

MySQL UPDATE

The UPDATE statement is used to modify the existing records in a table.


Syntax:


                    UPDATE table_name
                    SET column1 = value1, column2 = value2
                    WHERE condition;

Example:


                    UPDATE users
                    SET age = 35
                    WHERE name = 'John Doe';

MySQL DELETE

The DELETE statement is used to remove records from a table.


Syntax:


                    DELETE FROM table_name
                    WHERE condition;

Example:


                    DELETE FROM users
                    WHERE name = 'John Doe';

MySQL JOIN

JOIN is used to combine rows from two or more tables based on a related column between them.


Types of Joins:

  • INNER JOIN: Returns records that have matching values in both tables.
  • LEFT JOIN: Returns all records from the left table and the matched records from the right table.
  • RIGHT JOIN: Returns all records from the right table and the matched records from the left table.
  • FULL JOIN: Returns all records when there is a match in either left or right table.

Example:


                    SELECT users.name, orders.amount
                    FROM users
                    INNER JOIN orders ON users.id = orders.user_id;

MySQL WHERE Clause

The WHERE clause is used to filter records based on a given condition.


Example:


                    SELECT * FROM users
                    WHERE age > 30;

MySQL GROUP BY

The GROUP BY statement is used to group rows that have the same values into summary rows.


Example:


                    SELECT COUNT(*), city
                    FROM customers
                    GROUP BY city;

MySQL ORDER BY

The ORDER BY clause is used to sort the result set in either ascending or descending order.


Example:


                    SELECT * FROM users
                    ORDER BY name ASC;

MySQL Indexes

Indexes are used to speed up the retrieval of data from a table by creating a quick lookup method.


Example:


                    CREATE INDEX idx_name ON users (name);

MySQL Transactions

Transactions in MySQL are used to ensure that a group of SQL operations are executed successfully together.


Example:


                    START TRANSACTION;
                    UPDATE accounts SET balance = balance - 100 WHERE user_id = 1;
                    UPDATE accounts SET balance = balance + 100 WHERE user_id = 2;
                    COMMIT;

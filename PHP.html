PHP Introduction

  • PHP is a popular general-purpose scripting language that is especially suited to web development.
  • PHP was created by Rasmus Lerdorf in 1993 and is an open-source language.
  • It can be embedded into HTML, making it powerful for building dynamic web pages.

Why Learn PHP?

  • Server-Side Scripting: PHP is primarily used for server-side scripting, allowing for dynamic web page generation.
  • Large Community and Resources: PHP has extensive community support, tutorials, and documentation.
  • Compatibility: PHP works well with a variety of databases, especially MySQL, and supports many different servers.
  • Open Source: PHP is free to use, and there are many free libraries and frameworks available (such as Laravel, Symfony, etc.).

PHP Data Types

PHP supports several data types, which can be categorized as scalar types, compound types, and special types.


Scalar Types:

  • int: Represents integers.
  • float: Represents floating-point numbers.
  • string: Represents sequences of characters.
  • bool: Represents boolean values (true or false).

Compound Types:

  • array: Represents a collection of elements.
  • object: Represents instances of classes.

Special Types:

  • NULL: Represents a variable with no value.

Example:


                    $name = "John"; // string
                    $age = 25; // int
                    $isStudent = true; // bool
                    $height = 5.9; // float

PHP Operators

PHP provides a variety of operators for performing different operations on variables and values.


Arithmetic Operators:

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division
  • %: Modulus

Comparison Operators:

  • ==: Equal to
  • !=: Not equal to
  • >: Greater than
  • <: Less than
  • ===: Identical (value and type)

Logical Operators:

  • &&: Logical AND
  • ||: Logical OR
  • !: Logical NOT

PHP Booleans

A Boolean value can be either true or false. It is often used in conditional statements to determine the flow of the program.


Example:


                    $isAdult = true;
                    if ($isAdult) {
                        echo "You are an adult.";
                    } else {
                        echo "You are not an adult.";
                    }

PHP If ... Else

The if statement is used to check a condition and execute a block of code if the condition is true.


Example:


                    $age = 18;
                    if ($age >= 18) {
                        echo "You are an adult.";
                    } else {
                        echo "You are a minor.";
                    }

PHP While Loop

The while loop in PHP runs as long as the specified condition is true.


Example:


                    $i = 0;
                    while ($i < 5) {
                        echo "Number: $i 
";
                        $i++;
                    }

PHP For Loop

The for loop is used when the number of iterations is known beforehand.


Example:


                    for ($i = 0; $i < 5; $i++) {
                        echo "Number: $i 
";
                    }

PHP Break and Continue

break stops the loop, and continue skips to the next iteration.


Example:


                    for ($i = 0; $i < 10; $i++) {
                        if ($i == 5) {
                            break;
                        }
                        echo "Number: $i 
";
                    }

PHP Arrays

Arrays in PHP are used to store multiple values in a single variable.


Indexed Arrays:


                    $fruits = array("apple", "banana", "cherry");
                    echo $fruits[0];  // Output: apple

Associative Arrays:


                    $age = array("John" => 25, "Alice" => 22, "Bob" => 30);
                    echo $age["John"];  // Output: 25

PHP Strings

Strings in PHP are sequences of characters enclosed in either single or double quotes.


Example:


                    $greeting = "Hello, World!";
                    echo $greeting;  // Output: Hello, World!

PHP User Input

PHP allows you to get user input from forms using the $_GET and $_POST superglobals.


Example - Using $_GET:


                    $name = $_GET["name"];
                    echo "Hello, " . $name . "!";

PHP Functions

Functions in PHP are blocks of code that perform a specific task and can be reused.


Example:


                    function greet($name) {
                        echo "Hello, " . $name;
                    }
                    greet("John");  // Output: Hello, John

PHP Dictionaries

In PHP, dictionaries are implemented as associative arrays where keys are mapped to values.


Example:


                    $person = array("name" => "John", "age" => 30);
                    echo $person["name"];  // Output: John

PHP File I/O

PHP provides several functions to read from and write to files.


Example - Reading a File:


                    $file = fopen("file.txt", "r");
                    $content = fread($file, filesize("file.txt"));
                    fclose($file);
                    echo $content;

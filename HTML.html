HTML Introduction


HTML (Hypertext Markup Language) is the standard language for creating web pages and web applications. It structures content using a series of elements enclosed in tags. Web browsers interpret HTML to render and display the content on a page.


Every website you visit is made up of HTML documents. While HTML defines the structure, it does not handle the visual style or interactivity. These are handled by CSS and JavaScript respectively.


Key Points:

  • HTML is used to define the structure and content of a webpage.
  • It consists of tags that mark the beginning and end of content.
  • HTML is the backbone of any website and must be combined with other technologies like CSS and JavaScript for full functionality.

HTML Basic Structure


The basic structure of an HTML document involves several key elements that help define the webpage.


Basic HTML Template:


                <!DOCTYPE html>
                <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>Page Title</title>
                    </head>
                    <body>
                        <h1>Welcome to HTML!</h1>
                        <p>This is your first webpage.</p>
                    </body>
                </html>

Each HTML document begins with a <!DOCTYPE html> declaration that defines the document type. The <html> tag encloses the entire document. Inside the <head>, you place metadata such as the title and character encoding. The <body> contains the visible content of the webpage.


HTML Elements


HTML elements are the building blocks of a webpage. An element consists of an opening tag, content, and a closing tag. The opening tag marks the start of the element, and the closing tag marks the end.


Example:


                <p>This is a paragraph of text.</p>
                <h1>This is a heading</h1>

The above examples show two HTML elements: a paragraph and a heading. The <p> and <h1> tags indicate where the content starts and ends. There are many different types of HTML elements used to define various kinds of content, including text, links, images, and more.


Self-Closing Tags:


Some elements, such as images or line breaks, do not have closing tags. These are known as self-closing tags.



                <br>  
                <img src="image.jpg" alt="Description">

HTML Attributes


Attributes provide additional information about an element. They are always included in the opening tag and follow a name-value pair format: name="value".


Common HTML Attributes:

  • id: Uniquely identifies an element on the page (e.g., <div id="header">).
  • class: Assigns one or more classes to an element, which can be used for styling (e.g., <p class="intro">).
  • href: Defines the destination of a hyperlink (e.g., <a href="https://example.com">Link</a>).
  • style: Adds inline CSS to an element (e.g., <p style="color: blue;">).
  • src: Defines the source of an image or media (e.g., <img src="image.jpg">).

Attributes allow you to customize how elements behave and appear on the page.


HTML Headings


Headings are used to define the titles or sections of content in your webpage. HTML provides six levels of headings, ranging from <h1> (the largest) to <h6> (the smallest).


Example:


                <h1>Main Heading</h1>
                <h2>Subheading</h2>
                <h3>Smaller Subheading</h3>

Headings are important for structuring the content of your webpage and for search engine optimization (SEO) because they help search engines understand the hierarchy and relevance of the content.


HTML Paragraphs


Paragraphs in HTML are defined using the <p> tag. They are used to structure blocks of text on a webpage.


Example:


                <p>This is a paragraph of text. Paragraphs help break up long sections of text and make content more readable.</p>

Each paragraph will appear with space above and below it, as browsers automatically apply margins to paragraphs.


HTML Styles


CSS (Cascading Style Sheets) is used to style HTML elements. Styles can be added directly to HTML elements using the style attribute or through internal or external CSS.


Inline Style Example:


                <p style="color: red;">This paragraph is red.</p>

Inline styles are useful for quick changes but are not recommended for large websites as they can make the code harder to maintain.


Internal Style Example:


                <style>
                    p { color: blue; font-size: 16px; }
                </style>

Internal styles are written within the <head> section of the HTML document and can be used to style the whole page.


HTML Text Formatting


HTML provides various tags for formatting text. These formatting elements help to emphasize, highlight, or change the appearance of text content on the page.


Common Text Formatting Tags:

  • <b>: Bold text - Makes the enclosed text bold.
  • <i>: Italic text - Makes the enclosed text italicized.
  • <u>: Underlined text - Underlines the enclosed text.
  • <s>: Strikethrough text - Draws a line through the text.
  • <mark>: Highlighted text - Highlights text with a background color.
  • <small>: Smaller text - Displays smaller text.
  • <sub>: Subscript text - Renders text below the baseline.
  • <sup>: Superscript text - Renders text above the baseline.

Examples:


                <b>This is bold text</b>
                <i>This is italic text</i>
                <u>This is underlined text</u>
                <s>This is strikethrough text</s>
                <mark>This text is highlighted</mark>
                <small>This text is smaller</small>
                <sub>This is subscript</sub>
                <sup>This is superscript</sup>

These tags can be used individually or combined to create different effects in your text.


Combining Formatting Tags:


You can also combine multiple formatting tags to apply more than one style to the same content.



                <b><i>This is bold and italicized text</i></b>

This would render the text as both bold and italicized.


HTML Quotation


In HTML, quotations are used to indicate text that is a quotation or to attribute speech to someone. HTML provides several elements for this purpose.


Blockquote:


The <blockquote> element is used to indicate a block of text that is a quotation from another source. It is typically displayed with indented margins.



                <blockquote>
                    <p>This is a blockquote example. A blockquote is used to display text quoted from another source.</p>
                </blockquote>

Cite:


The <cite> element is used to reference the source of a quotation or work.



                <cite>Shakespeare, William. Hamlet. Act 3, Scene 2</cite>

Inline Quotation:


For inline quotations (a short quote within a paragraph), use the <q> element. It typically adds quotation marks around the quoted text.



                <p>The famous quote <q>To be, or not to be</q> is from Shakespeare's Hamlet.</p>

HTML Comments


Comments in HTML are used to add notes or explanations within the code that will not be displayed in the browser. Comments are often used to document code or to temporarily disable parts of code while testing or debugging.


Syntax of HTML Comments:


                <!-- This is a comment and will not be displayed in the browser -->

Comments can be placed anywhere within the HTML document, except within the <style> and <script> elements.


Example:


                <!-- This is a header section -->
                <header>
                    <h1>Welcome to the HTML Tutorial</h1>
                </header>

HTML Colors


Colors can be applied to HTML elements using CSS. You can specify colors using color names, hexadecimal values, RGB values, or RGBA for transparency.


Color Names:


HTML supports 140 named colors. For example, red, green, and blue.



                <p style="color: red;">This text is red.</p>

Hexadecimal Colors:


Hexadecimal values consist of a hashtag followed by six digits (e.g., #FF5733). The first two digits represent red, the next two represent green, and the last two represent blue.



                <p style="color: #FF5733;">This text is a custom orange color.</p>

RGB Colors:


RGB (Red, Green, Blue) values allow for more precise color control. The format is rgb(r, g, b), where each value ranges from 0 to 255.



                <p style="color: rgb(255, 87, 51);">This text is a custom RGB orange color.</p>

RGBA Colors:


RGBA is similar to RGB, but it also includes an alpha channel (transparency), which ranges from 0 (fully transparent) to 1 (fully opaque).



                <p style="color: rgba(255, 87, 51, 0.5);">This text is a semi-transparent orange.</p>

HTML Styles - CSS


CSS (Cascading Style Sheets) is used to style HTML elements. While HTML provides the structure, CSS controls the visual appearance of the page, such as colors, fonts, layouts, and spacing.


Types of CSS


There are three ways to apply CSS to HTML:

  • Inline CSS: Applied directly to individual HTML elements using the style attribute.
  • Internal CSS: Defined within the <style> tag inside the <head> section of the HTML document.
  • External CSS: Defined in an external stylesheet, which is linked to the HTML document using the <link> tag.

Inline CSS Example:



                <p style="color: blue; font-size: 18px;">This is a paragraph with inline CSS.</p>

This example applies inline CSS directly to the <p> element, changing its text color to blue and its font size to 18px.


Internal CSS Example:


Internal CSS is placed within the <style> tag in the document's <head> section:



                <style>
                    p {
                        color: green;
                        font-size: 20px;
                    }
                </style>

This will apply the green color and 20px font size to all <p> elements in the document.


External CSS Example:


External CSS is defined in a separate file, and linked to the HTML document using the <link> tag:



                <link rel="stylesheet" href="styles.css">

In the "styles.css" file, you can write your CSS rules:



                p {
                    color: red;
                    font-size: 22px;
                }

This way, all <p> elements across multiple HTML pages can have consistent styling.


CSS Selectors


CSS selectors allow you to target specific HTML elements and apply styles to them. Some common selectors are:

  • Type Selector: Targets elements by their tag name (e.g., p, h1, div).
  • Class Selector: Targets elements with a specific class attribute (e.g., .myClass).
  • ID Selector: Targets elements with a specific ID attribute (e.g., #myID).
  • Universal Selector: Targets all elements in the document (e.g., *).

Example of Class Selector:


In HTML, you apply a class to an element:



                <div class="container">This is a div with class "container"</div>

In CSS, you target the class:



                .container {
                    background-color: lightgray;
                    padding: 10px;
                }

This will apply a light gray background and padding to the div with the class "container".


CSS Box Model


The CSS box model defines how the elements on a page are structured and how their size is calculated. It consists of four parts:

  • Content: The actual content of the element (e.g., text, images).
  • Padding: The space between the content and the border.
  • Border: The border around the padding (if any).
  • Margin: The space outside the border, separating the element from other elements.

By adjusting the padding, border, and margin, you can control the spacing and layout of your page.


Example of Box Model:



                div {
                    width: 200px;
                    padding: 10px;
                    border: 5px solid black;
                    margin: 20px;
                }

This will create a div with a width of 200px, 10px padding, a 5px black border, and 20px margin.


Responsive Web Design


Responsive web design ensures that your web pages look good on all devices, from desktops to mobile phones. This is achieved by using flexible layouts, grids, and media queries.


Media Queries:


Media queries allow you to apply different styles depending on the screen size of the device. Here's an example:



                @media (max-width: 768px) {
                    body {
                        font-size: 14px;
                    }
                }

This CSS rule changes the font size to 14px on screens that are 768px wide or smaller (like tablets or mobile phones).


CSS Flexbox


Flexbox is a layout module in CSS that allows you to create flexible and responsive layouts with ease. It distributes space along a row or column.


Example of Flexbox Layout:



                .container {
                    display: flex;
                    justify-content: space-between;
                }

                .item {
                    width: 100px;
                    height: 100px;
                    background-color: lightblue;
                    margin: 10px;
                }

This will create a flex container with items spaced evenly along the row, each item having a width of 100px and a background color of light blue.


HTML Links


Links are an essential part of the web, allowing users to navigate between pages or resources. In HTML, links are created using the <a> tag. The <a> tag stands for anchor and is used to define hyperlinks.


Creating a Basic Link


To create a basic hyperlink, use the <a> tag with the href attribute, which specifies the URL or destination of the link:



                <a href="https://www.example.com">Visit Example</a>

This will display a clickable link with the text "Visit Example" that will take the user to the specified URL when clicked.


Using the Target Attribute


The target attribute specifies where to open the linked document. By default, links open in the same window. You can change this by setting the target to _blank to open the link in a new tab or window:



                <a href="https://www.example.com" target="_blank">Open in a new tab</a>

Using target="_blank" ensures the link opens in a new browser tab, keeping the user on the current page.


Links to Other Sections on the Same Page


You can also create links that navigate to specific sections within the same page using anchor links. This is done by adding an id attribute to the target element and linking to it using the href attribute:



                <a href="#section2">Go to Section 2</a>

In this example, clicking the link will scroll the page down to the element with id="section2".


Links to External Documents


HTML links can also point to external documents, such as PDFs or Word documents. For instance, you can link to a PDF file like this:



                <a href="document.pdf">Download the PDF</a>

When the user clicks the link, the PDF file will either open in the browser (depending on the browser's settings) or prompt the user to download it.


Adding Link Titles


Use the title attribute to provide additional information about the link. This will display when the user hovers over the link:



                <a href="https://www.example.com" title="Visit Example Website">Visit Example</a>

The title attribute provides a tooltip when the user hovers over the link, enhancing accessibility and providing more context to the link.


Linking to Email Addresses


In addition to linking to websites, you can also create mailto links that open the user's default email client with a new message composed. This is done using the mailto: scheme in the href attribute:



                <a href="mailto:someone@example.com">Send Email</a>

Clicking this link will open the default email application with a new message addressed to someone@example.com.


Links to Telephone Numbers


Similarly, you can create links to telephone numbers using the tel: scheme. This is especially useful for mobile users who can click a link to call a phone number directly:



                <a href="tel:+1234567890">Call Us</a>

Clicking this link on a mobile device will initiate a call to the provided number.


Linking to Files on Your Website


If you want to link to a file hosted on your website, simply use a relative path instead of a full URL. For example, if you have an image called logo.png in your website's root directory:



                <a href="logo.png">View Logo</a>

This will open the image when clicked, as long as the image is in the same directory as the HTML file.


Best Practices for Links


Here are a few best practices to consider when working with links:

  • Use descriptive anchor text that tells the user what they will find when they click the link.
  • Make sure links are easily recognizable (e.g., use underlines or color contrasts).
  • Always test links to ensure they work correctly and lead to the intended destination.
  • Provide context for links that may open new windows or tabs, so users know what to expect.

HTML Images


Images are an essential part of modern web design, used to enhance content and provide visual appeal. In HTML, you can embed images using the <img> tag.


Basic Image Syntax


To display an image, use the <img> tag with the src attribute, which specifies the path to the image file. The alt attribute provides alternative text for the image if it can't be displayed:



                <img src="image.jpg" alt="Description of the image">

In this example, the image located at image.jpg will be displayed. If the image is not found, the alternative text "Description of the image" will be shown instead.


Image Attributes


The <img> tag supports several attributes:

  • src: Specifies the image file location (can be a relative or absolute path).
  • alt: Provides alternative text for the image, improving accessibility.
  • width and height: Specifies the dimensions of the image (in pixels).
  • title: Displays a tooltip when the user hovers over the image.

Example of Resizing an Image


To resize an image, use the width and height attributes:



                <img src="image.jpg" alt="Description" width="300" height="200">

This will display the image at 300px width and 200px height, regardless of the original image dimensions.


Linking an Image


Just like text, images can also be linked to other pages or resources. To do this, simply wrap the <img> tag inside an <a> tag:



                <a href="https://www.example.com">
                    <img src="logo.png" alt="Example Logo">
                </a>

This will make the image clickable, and when clicked, it will redirect the user to the specified URL.


Example of an Image on the Page


Below is an example of an image displayed using the <img> tag. The image is sourced from a relative path and includes the alt and title attributes for accessibility and enhanced user experience:



                <img src="https://www.example.com/logo.png" alt="Example Logo" title="Click to Visit Example" width="250">

This will show the "Example Logo" image on the page. The alt text ensures that users with visual impairments or those using screen readers can understand the image content.


Embedding Images from External Sources


You can also embed images from external websites using the image URL. Here is an example of displaying an image hosted on an external server:



                <img src="https://www.example.com/images/external-logo.png" alt="External Logo" width="300">

This will fetch the image from the provided external URL and display it on your page.


Best Practices for Using Images

  • Use descriptive alt text for accessibility, especially for visually impaired users.
  • Optimize image file sizes to improve website loading speed.
  • Use responsive image techniques (such as srcset) to ensure images display correctly on different screen sizes.
  • Always provide a fallback for images in case they fail to load (using the alt attribute).
  • Ensure the image file format is appropriate for the content (e.g., JPEG for photos, PNG for graphics with transparency, SVG for scalable vector images).

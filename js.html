JavaScript Introduction

  • JavaScript (JS) is a high-level, interpreted scripting language that enables dynamic content on websites.
  • It is one of the core technologies of web development, alongside HTML and CSS.
  • JavaScript is versatile and used in client-side as well as server-side development (via Node.js).

Why Learn JavaScript?

  • Interactivity: JavaScript adds interactivity to web pages, such as form validation, animations, and dynamic content loading.
  • Wide Usage: It's used for both front-end (in the browser) and back-end development (via Node.js).
  • Versatile: JavaScript can be used for various applications, from websites to mobile apps (React Native), and even server-side scripting (Node.js).
  • Huge Community: JavaScript has a massive community, with many frameworks, libraries, and tools available to simplify development.

Basic Syntax of JavaScript


                // Declaring variables
                let name = 'CodeMastery';
                const year = 2024;

                // A simple function to greet
                function greet() {
                    console.log("Welcome to JavaScript!");
                }
                
                greet();  // Calling the function

Data Types in JavaScript

JavaScript has different data types to represent different types of values:


Basic Data Types:

  • String: Represents a sequence of characters.
  • Number: Represents both integer and floating-point numbers.
  • Boolean: Represents a true/false value.
  • undefined: Represents a variable that has been declared but not assigned a value.
  • null: Represents the intentional absence of any value.
  • Symbol: Represents a unique and immutable value.

Example:


                let age = 25; // Number
                let isActive = true; // Boolean
                let name = "John"; // String
                let user = null; // null value
                let address; // undefined

Special Note on Types:

  • JavaScript is a loosely-typed language, which means you don't need to explicitly declare the type of variable.

JavaScript Operators

Operators are used to perform operations on variables and values in JavaScript.


Types of Operators:

  • Arithmetic Operators: +, -, *, /, %, ++, --
  • Assignment Operators: =, +=, -=, *=, /=
  • Comparison Operators: ==, ===, !=, !==, >, <, >=, <=
  • Logical Operators: &&, ||, !

Example:


                let x = 10;
                let y = 5;

                // Arithmetic Operation
                let sum = x + y;  // 15

                // Comparison
                let isEqual = (x == y);  // false

                // Logical
                let result = (x > 5 && y < 10); // true

JavaScript Functions

Functions in JavaScript allow you to encapsulate code into reusable blocks.


Basic Function Syntax:


                function greetUser(name) {
                    console.log("Hello, " + name);
                }

                greetUser("Alice");  // Outputs: Hello, Alice

Function with Return:


                function sum(a, b) {
                    return a + b;
                }

                let result = sum(5, 10);  // result = 15
                console.log(result);

Arrow Function:


                const multiply = (a, b) => a * b;

                console.log(multiply(4, 5));  // Outputs: 20

JavaScript If ... Else

Use `if` statements to execute code based on conditions.


Example:


                let age = 18;

                if (age >= 18) {
                    console.log("You are an adult.");
                } else {
                    console.log("You are a minor.");
                }


JavaScript Switch Statement

The `switch` statement is used to execute different blocks of code based on a condition.


Syntax:


                switch (expression) {
                    case value1:
                        // block of code
                        break;
                    case value2:
                        // block of code
                        break;
                    default:
                        // block of code
                }


JavaScript Loops

Loops are used to repeat a block of code multiple times, as long as the specified condition is true.


Types of Loops in JavaScript:

  • for loop: Executes a block of code a specific number of times.
  • while loop: Executes a block of code as long as the condition is true.
  • do...while loop: Executes a block of code at least once and then repeats it as long as the condition is true.

Example of a `for` loop:


                for (let i = 0; i < 5; i++) {
                    console.log(i);  // Outputs 0, 1, 2, 3, 4
                }

Example of a `while` loop:


                let i = 0;
                while (i < 5) {
                    console.log(i);  // Outputs 0, 1, 2, 3, 4
                    i++;
                }

Example of a `do...while` loop:


                let i = 0;
                do {
                    console.log(i);  // Outputs 0, 1, 2, 3, 4
                    i++;
                } while (i < 5);

JavaScript Arrays

Arrays are used to store multiple values in a single variable.


Array Syntax:


                let fruits = ["apple", "banana", "orange"];
                console.log(fruits[0]);  // Outputs: apple

Array Methods:

  • push(): Adds one or more elements to the end of an array.
  • pop(): Removes the last element from an array.
  • shift(): Removes the first element from an array.
  • unshift(): Adds one or more elements to the beginning of an array.
  • length: Returns the number of elements in an array.

Example of Array Methods:


                let fruits = ["apple", "banana", "orange"];
        
                fruits.push("grape");  // Adds "grape" to the end of the array
                console.log(fruits);  // ["apple", "banana", "orange", "grape"]
        
                fruits.pop();  // Removes the last element ("grape")
                console.log(fruits);  // ["apple", "banana", "orange"]

JavaScript Objects

Objects are collections of properties and methods. A property is a key-value pair, and a method is a function inside an object.


Object Syntax:


                let person = {
                    firstName: "John",
                    lastName: "Doe",
                    age: 30,
                    greet: function() {
                        console.log("Hello, " + this.firstName + " " + this.lastName);
                    }
                };
                person.greet();  // Outputs: Hello, John Doe

Accessing Object Properties:

  • Using dot notation: `person.firstName`
  • Using bracket notation: `person["firstName"]`

JavaScript Events

Events are actions that occur in the browser (e.g., a user clicks a button, submits a form, etc.). JavaScript allows you to listen for events and execute code in response.


Common Events:

  • click: Triggered when an element is clicked.
  • keydown: Triggered when a key is pressed down.
  • submit: Triggered when a form is submitted.
  • mouseover: Triggered when the mouse pointer hovers over an element.

Event Listener Syntax:


                let button = document.querySelector("button");
        
                button.addEventListener("click", function() {
                    alert("Button clicked!");
                });

DOM Manipulation in JavaScript

The DOM (Document Object Model) allows JavaScript to access and manipulate HTML elements dynamically.


Common DOM Methods:

  • getElementById(): Selects an element by its ID.
  • querySelector(): Selects the first element that matches a CSS selector.
  • createElement(): Creates a new HTML element.
  • appendChild(): Appends a new child element to an existing element.

Example of DOM Manipulation:


                let div = document.createElement("div");
                div.textContent = "This is a new div!";
                
                document.body.appendChild(div);  // Adds the new div to the body

Asynchronous JavaScript

Asynchronous JavaScript allows for non-blocking code execution, enabling tasks like fetching data from an API without freezing the browser.


Common Asynchronous Methods:

  • setTimeout(): Executes a function after a specified delay.
  • setInterval(): Repeats a function at specified intervals.
  • fetch(): Makes an HTTP request to fetch data asynchronously.

Example with `setTimeout()`:


                setTimeout(function() {
                    console.log("This message appears after 3 seconds");
                }, 3000);  // 3000ms = 3 seconds

Example with `fetch()`:


                fetch("https://api.example.com/data")
                    .then(response => response.json())
                    .then(data => console.log(data))
                    .catch(error => console.error("Error:", error));

JavaScript Error Handling

JavaScript provides a way to handle errors and prevent your program from crashing by using `try...catch` blocks.


Basic Syntax of `try...catch`:


                try {
                    // Code that may throw an error
                    let result = someFunction();
                } catch (error) {
                    // Code that runs if an error occurs
                    console.log("Error: " + error.message);
                }

Example:


                try {
                    let number = 10;
                    let result = number / 0;  // This will cause an error (infinity)
                    console.log(result);
                } catch (error) {
                    console.log("Error: Division by zero is not allowed.");
                }
